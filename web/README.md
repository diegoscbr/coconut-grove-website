# CGSC Instructional Center — website

Plain Next.js (App Router) site. All copy lives in typed content files in
`lib/content/*` — no CMS, no env vars, fully static.

See **[SETUP.md](./SETUP.md)** for how to run it locally, edit content, and
deploy. `HANDOFF.md` tracks remaining pre-production items.

```bash
npm install
npm run dev   # http://localhost:3000
```

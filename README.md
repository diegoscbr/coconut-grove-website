# Coconut Grove Sailing Club — Instructional Center website

Website redesign for the CGSC Instructional Center (cgscic.org): a plain
Next.js (App Router) site. All copy lives in typed content files in
`lib/content/*` — no CMS, no env vars, every page statically prerendered.

```bash
npm install
npm run dev   # http://localhost:3000
```

- **[SETUP.md](./SETUP.md)** — run locally, edit content, deploy
- **[HANDOFF.md](./HANDOFF.md)** — remaining pre-production items
- `design/`, `reference/`, `BRAND_DECISIONS.md` — design history and brand
  source material (not part of the build)
